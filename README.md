# 🚀 InterviewAI – AI-Powered Mock Interview Platform

An interactive web application that simulates real-world technical and behavioral interviews using Generative AI. It helps users practice coding interviews, improve problem-solving skills, and receive intelligent feedback.

---

## 🌐 Live Demo

🔗 https://interviewai-0yw3.onrender.com

---

## 📌 Features

* 🤖 AI-powered mock interviews (technical + behavioral)
* 🧠 Dynamic question generation using OpenAI / Gemini APIs
* 💻 Real-time coding interview simulation
* 📊 Performance evaluation (time & space complexity insights)
* ⚡ Fast and responsive UI with seamless user experience

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Anshu404/InterviewAI.git
cd InterviewAI
```

### 2️⃣ Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file in the server folder and add:

```env
OPENAI_API_KEY=your_key
GEMINI_API_KEY=your_key
RAZORPAY_KEY=your_key
```

### 4️⃣ Run the project

```bash
# Run backend
cd server
npm start

# Run frontend
cd client
npm run dev
```
